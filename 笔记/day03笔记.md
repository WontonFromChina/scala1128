## 2. 高阶函数

参数有函数或者返回值是函数的函数就是高阶函数!!!

### 2.1 匿名函数

定义: 没有名字的函数, 就叫匿名函数

作用:  一般用于向高阶函数传递参数

返回值类型: 一般开推导出来.

重要: 用于不要在匿名函数中使用 `return`

用法:

1. 作为值存储在变量中

   ```scala
   val f = (a: Int) => a + 10
   val f1 = (a: Int, b:Int) => a + b
   val f2 = () => println(".....")
   ```

2. 给高阶函数传值

   ```scala
   // 定义高阶函数
   def foo(op: (Int, Int) => Int) = op(10, 20)
   
   val r = foo( (a: Int, b:Int) => a + b  )  // 30
   
   val r1 = foo( (a, b) => a + b )
   
   val r2 = foo( _ + _ )
   ```

### 2.2 高阶函数的具体使用

### 2.4 柯里化

```scala
def add(a:Int, b:Int) = a + b

def add(a:Int)(b:Int) = a + b
```

>  定义: 将一个参数列表的多个参数变成多个参数列表的过程, 就叫对函数柯里化(currying)

> 具体应用:  柯里化一般只会柯里化出来两个参数列表, 
>
> 第一个参数列表一般是必须要传入的参数
>
> 第二个参数列表一般是定义隐式参数, 将来可以传, scala会自动找隐式值去传.

理论基础是闭包!!!
